
✊✋✌️ Rock-Paper-Scissors Game (Java)

A simple console-based Rock-Paper-Scissors game developed in Java. In this game, the user competes against the computer, which makes random choices. The winner is determined based on the standard Rock-Paper-Scissors rules.

## 📌 Features

- Play against the computer
- Random computer choices using the `Random` class
- Menu-driven gameplay
- Play multiple rounds
- Input validation
- Displays the winner after each round

## 🛠️ Technologies Used

- Java
- Scanner Class
- Random Class
- Arrays
- Loops
- Conditional Statements
- Switch Case (if used)

## 📂 Project Structure

```
RockPaperScissors.java
```

## 🚀 How to Run

1. Clone this repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile the program:
   ```
   javac RockPaperScissors.java
   ```
4. Run the program:
   ```
   java RockPaperScissors
   ```

## 🎮 Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- If both players choose the same option, the game is a draw.
